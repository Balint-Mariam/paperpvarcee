# Outputs

External-facing outputs are organized as follows:

- `01_model_ready_data/`: final balanced model-ready dataset and transformation checks.
- `02_tables/main_paper/`: tables intended for the main manuscript.
- `02_tables/appendix/`: appendix tables.
- `02_tables/robustness/`: robustness and diagnostics outputs.
- `03_figures/main_paper/`: main manuscript figures.
- `03_figures/appendix/`: appendix figures.
- `04_reports/`: readable reports and captions.
- `05_logs/`: run logs and reproducibility checks.
- `06_replication_only/`: old, exhaustive or unpolished outputs retained for traceability only.
