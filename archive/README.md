# Archive

This folder contains internal or superseded material retained for traceability:

- `internal_audit/`: full audit workbooks, previous handoff outputs and code snapshots.
- `old_master_scripts/`: archived legacy consolidation script.
- `old_outputs/`: local ignored intermediate caches and old outputs.
- `old_figures/`: reserved for superseded figure sets.
- `previous_model_variants/`: reserved for earlier model variants not used in the final manuscript.

External readers should start with `README.md`, `docs/OUTPUT_ROADMAP.md` and `outputs/`.
